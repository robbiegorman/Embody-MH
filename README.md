# Embody Mental Health Consultation Landing Page

This repository contains a single-page static website for the Embody Mental Health trauma-informed consultation offering.

## Files

- `index.html` — the full website code
- `.nojekyll` — helps GitHub Pages serve the site cleanly
- `README.md` — this file

## Publish with GitHub Pages

1. Upload `index.html`, `.nojekyll`, and `README.md` to the root of a GitHub repository.
2. Go to **Settings → Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Choose `main` and `/root`.
5. Save and wait a few minutes.

Your public site will be available at a URL like:

`https://yourusername.github.io/repository-name/`

## Form note

GitHub Pages does not process forms. To make the forms work, connect them to Formspree, Tally, HubSpot, Google Forms, or host on Netlify and enable Netlify Forms.
