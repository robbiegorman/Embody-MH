# Embody Mental Health Landing Page

This is a single-page static website for Embody Mental Health. It is ready to publish with GitHub Pages.

## Files

- `index.html` — the full website code
- `.nojekyll` — tells GitHub Pages to serve the site as a plain static HTML site

## How to publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `.nojekyll`, and this `README.md` file to the repository.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
5. Click **Save**.
6. GitHub will create a live link, usually in this format:
   `https://YOUR-USERNAME.github.io/YOUR-REPOSITORY-NAME/`

## Before sharing publicly

Replace placeholder content in `index.html`, especially:

- Consultant bio/credentials
- Contact/scheduling information
- Photo or brand graphic placeholder
- Any legal disclaimer language you want reviewed by counsel
- Checklist download process or form handling

## Notes about forms

This file includes simple HTML forms. GitHub Pages does not process form submissions by itself. To collect submissions, connect the forms to a service such as Formspree, Basin, HubSpot, Tally, Google Forms, or another form endpoint.
